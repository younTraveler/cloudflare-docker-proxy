为了加速 Docker 镜像拉取，你可以使用以下命令设置 registry mirror:

sudo tee /etc/docker/daemon.json <<EOF
{
    "registry-mirrors": ["https://{{host}}"]“registry-mirrors”:(" https://{{主机}}”)
}
EOF
# 配置完后需要重启 Docker 服务
sudo systemctl restart docker执行命令systemctl restart docker
